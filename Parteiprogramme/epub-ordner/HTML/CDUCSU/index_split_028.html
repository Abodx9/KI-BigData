Keine neuen Schulden für Deutschland

• Deutschland soll keine neuen Schulden machen.

Manchmal möchte ein Land Geld ausgeben.

Das Land hat aber nicht so viel Geld.

Dann nimmt das Land Schulden auf.

Das heißt: Das Land leiht sich Geld aus.

Auch Deutschland hat Schulden.

Wir finden: Die Schulden von Deutschland sollen weniger werden.

So schnell wie möglich.

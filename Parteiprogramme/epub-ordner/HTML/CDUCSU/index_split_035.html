Seite 19

 

So soll das gehen:

Wenn man sich ein Haus oder eine Wohnung kauft, dann muss man auch Steuern zahlen.

Das heißt:

Man muss dem Staat Geld geben.

Dazu sagt man auch:

Grund-Erwerbs-Steuer.

Wir wollen die Gesetze so ändern:

Dass man dem Staat

weniger Grund-Erwerbs-Steuer zahlen muss.
